# Gstream3-converter

### Command
```
# grid to edge list
./dense_to_el32 <GRID_NAME> <INPUT_PATH> <OUTPUT_PATH> <OUTPUT_BUF_SIZE:MB> <IS_DENSE_FORMAT>

# grid to optimal edge list
./grid_to_oel32 <GRID_NAME> <INPUT_PATH> <OUTPUT_PATH> <OUTPUT_BUF_SIZE:MB> <IS_DENSE_FORMAT>

# edge list to optimal edge list
./el32_to_oel32 <RMAT_SIZE> <INPUT_PATH> <OUTPUT_PATH> <INPUT_BUF_SIZE:MB> <OUTPUT_BUF_SIZE:MB> <IS_BIG_ENDIAN>
```
